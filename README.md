## Requerimentos

- [x] NodeJS (para rodar o comando npm)

## Instalação

```sh
git clone https://github.com/manticorasolution/manticora-dashboard.git
cd manticora-dashboard
npm i
```

## Instruções de Uso

```sh
npm run start
```
